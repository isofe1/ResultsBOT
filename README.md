## خطوات التشغيل

### 1. إنشاء البوت على تيليجرام
1. افتح تيليجرام وابحث عن **@BotFather**
2. أرسل له: `/newbot`
3. اختر اسماً للبوت (مثلاً: نتائج ذي قار)
4. اختر username للبوت (مثلاً: `dhiqar_results_bot`)
5. ستحصل على **TOKEN** — احفظه جيداً

### 2. تثبيت Python
- حمّل Python من: https://python.org
- تأكد من تحديد خيار "Add to PATH" عند التثبيت

### 3. تثبيت المكتبات
افتح Terminal أو CMD في مجلد البوت وشغّل:
```bash
pip install python-telegram-bot==21.6
```

### 4. إعداد البوت
افتح ملف `bot.py` وغيّر هذا السطر:
```python
BOT_TOKEN = "ضع_توكن_البوت_هنا"
```
إلى:
```python
BOT_TOKEN = "1234567890:AABBccDDeeFFggHH..."  ← التوكن الذي حصلت عليه
```

### 5. تشغيل البوت
```bash
python bot.py
```

## الاستضافة (للتشغيل الدائم 24/7)

### الخيار 1 — VPS رخيص (موصى به)
- **Contabo**: ~4$ شهرياً (الأرخص)
- **DigitalOcean**: ~6$ شهرياً
- **Hetzner**: ~4$ شهرياً

بعد الحصول على VPS (Linux Ubuntu):
```bash
# رفع الملفات
scp bot.py students.db requirements.txt user@IP:/home/user/bot/

# على السيرفر
cd /home/user/bot
pip install -r requirements.txt

# تشغيل دائم في الخلفية
nohup python3 bot.py &

# أو باستخدام screen
screen -S bot
python3 bot.py
# Ctrl+A ثم D للخروج دون إيقاف البوت
```

3. عيّن متغير `BOT_TOKEN` في الإعدادات
